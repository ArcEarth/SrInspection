#Build 
##Clone and pull this repository
> git clone this_repo<br/>
> git submodule init DirectXTK<br/>
> git submodule update<br/>

##Platform/Compilier
* Windows 8.1+, best with Windows 10
* Visual Studio 2015 (VC++ 14), best with Update 3
